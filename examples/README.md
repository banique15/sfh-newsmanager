# Examples Directory

This directory contains **source of truth** files that define what good looks like for the Newsletter Manager agent.

## Structure

- **[articles/](articles/)** - Reference article data structures
  - Article templates (draft, published, with images)
  - Database schema examples
  
- **[prompts/](prompts/)** - AI prompt templates
  - Article generation prompts
  - Image generation prompts
  - Clarification and confirmation prompts
  
- **[responses/](responses/)** - Expected agent responses
  - Slack responses
  - Email responses
  - Confirmation flows (Yes/Cancel)
  
- **[user_requests/](user_requests/)** - Real-world request examples
  - Sample Slack messages
  - Sample email bodies
  - Edge case scenarios

## Purpose

These examples serve as:
- **Templates** for consistency across all agent operations
- **Reference data** for testing and validation
- **Training materials** to understand expected behaviors
- **Documentation** of proper tone, format, and structure

## Usage

When implementing tasks from [instructions/](../instructions/), refer to relevant examples here to ensure consistency with established patterns and standards.
